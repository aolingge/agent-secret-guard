# Code of Conduct

This project is meant to be useful, direct, and safe for people working on security-sensitive developer tooling.

Be respectful, stay on topic, and assume good intent when discussing false positives, missed detections, or release risks. Do not post real secrets, private infrastructure details, personal data, or exploit details in public threads.

Maintainers may edit, hide, or remove comments that expose sensitive information or make the project harder to use safely.
