# Support

Use GitHub issues for bugs, false positives, false negatives, documentation gaps, and rule requests.

Direct links:

- Bug report: <https://github.com/aolingge/agent-secret-guard/issues/new?template=bug_report.yml>
- Rule request: <https://github.com/aolingge/agent-secret-guard/issues/new?template=rule_request.yml>
- Agent task: <https://github.com/aolingge/agent-secret-guard/issues/new?template=agent_task.yml>
- Security policy: <https://github.com/aolingge/agent-secret-guard/security/policy>
- Remediation guide: <https://github.com/aolingge/agent-secret-guard/blob/main/docs/remediation.md>

Before opening an issue:

1. Update to the latest version.
2. Run with `--format json` if you need machine-readable output.
3. Replace real tokens, cookies, private paths, and private hostnames with fake values.

For sensitive security reports, follow `SECURITY.md` instead of opening a public issue.
